Ch3: Arrays, Logic, and Loops

  • Array: created by = [] or new Array()
    • typeOf = "object"
    • delete= to remove item value from array- does not remove space of now "undefined" item
    • Destructuring Arrays
      • const [x, y] = [1, 2];
      • ex/ [x, y] = [y, x];
    • Array Properties & Methods
      • .length - if you set array = to length < original, other items with be removed
      • pop(), shift(), unshift(), push()
      • concat(), spread operator: ... can also be used to flatten arrays
      • join(): separates array by "," by default
      • slice(), splice(): same as slice(), but actually removes items from array and returns it as a subarray
      • reverse(), sort() - change order of array permanately
      • indexOf()
      • include()
  • Sets - cannot include duplicate values
    • Do not have index notation
    • create = new Set()
    • list.add(num)
    • .size()
    • .has() -works faster than indexOf and includes methods
    • .delete()
    • .clear()= removes all values
    • can be converted into an array using spread operator or Array.from()
  • Weak Sets
    • object in set exists in set even if original reference removed
    • garbage-collection automatically removing items no longer required by the program
    • weak sets garbage collect any references to a 'dead object' w/ orginal reference removed
    • new WeakSet()
    • only non-primitive data types
    • cannot access list of values in weak set
  • Maps
    • key, value pairs
    • key can be of any type
    • use get()to retrieve values
    • new Map()
    • set() can be used to add key, value pairs
    • has(), size(), get(), delete(), clear()
    • can be converted to array using spread operator or Array.from()
  • Weak Maps
    • work the same as weak sets
    • new WeakMap()
    • can all same methods except size()
  • Logic
    • if/else
    • ternary operator
      • condition ? (//code to run if condition is true) : (//code to run if condition is false)
      • even shorter by placing the ternary operator inside a template string
    • switch()
    • while() loops
    • infinte loops
    • do while()
    • for() loops
    • looping over arrays
      • ES6 introduced an improved iterator function for arrays called a for-of loop that uses a slightly different syntax
      • ex/ for(const value of avengers){ console.log(value); }
      • needs to be "const"
    • looping over maps
      • .keys()
      • .values()
      • .entries()= access both key and value
      • CANNOT loop over weak maps
